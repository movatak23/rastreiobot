Política de Privacidade

Esta política descreve como o LoggZap Dashboard coleta, usa e protege as informações dos usuários da extensão Chrome e do site loggzap.com.br.

1. Quem somos

O LoggZap Dashboard é uma extensão Chrome desenvolvida e operada por Ronaldo Valério, com sede em Recife, Pernambuco, Brasil. Para contato: contato@loggzap.com.br

2. Dados coletados

O LoggZap Dashboard coleta apenas os dados estritamente necessários para o funcionamento da extensão:

  • Credenciais de acesso à loja Nuvemshop (token OAuth gerado pelo próprio usuário)
  • Store ID da loja Nuvemshop do usuário
  • Chave secreta de autenticação entre extensão e backend
  • Metas financeiras definidas pelo usuário (valores diário, semanal e mensal)
  • Data de início do período trial
  • Plano de licença ativo

Todos esses dados são armazenados localmente no navegador do usuário via chrome.storage.local. Nenhum dado pessoal é transmitido para servidores de terceiros.

3. Dados NÃO coletados

O LoggZap Dashboard não coleta, armazena nem transmite:

  • Nome, CPF, endereço ou qualquer dado pessoal do usuário
  • Dados financeiros ou de pagamento
  • Histórico de navegação
  • Conteúdo de outras abas ou sites visitados
  • Dados de clientes da loja do usuário
  • Senhas ou informações de autenticação de terceiros

4. Como os dados são usados

Os dados armazenados localmente são utilizados exclusivamente para:

  • Autenticar a extensão com o backend do LoggZap
  • Buscar os dados da loja Nuvemshop do usuário via API oficial
  • Exibir métricas, pedidos e metas no painel lateral da extensão
  • Validar e gerenciar o plano de licença ativo

5. Compartilhamento de dados

O LoggZap não vende, aluga nem compartilha dados dos usuários com terceiros. As únicas comunicações externas realizadas pela extensão são:

  • API Nuvemshop (api.nuvemshop.com.br) — para buscar pedidos e dados da loja do próprio usuário, usando o token OAuth fornecido pelo usuário
  • Backend LoggZap (railway.app) — para validação de licença e autenticação, usando apenas o Store ID e a chave secreta
  • WhatsApp Web (web.whatsapp.com) — apenas para abrir conversas pré-preenchidas quando o usuário solicita envio de notificação de pedido

6. Armazenamento e segurança

Todos os dados são armazenados localmente via chrome.storage.local, que é isolado por extensão e não acessível por outros sites ou extensões. A comunicação com o backend é realizada via HTTPS com autenticação por chave secreta.

O usuário pode excluir todos os dados armazenados pela extensão a qualquer momento desinstalando a extensão ou limpando os dados de armazenamento nas configurações do Chrome.

7. Dados de pagamento

Os pagamentos de assinaturas são processados pelo Mercado Pago. O LoggZap não armazena nem tem acesso a dados de cartão de crédito ou informações bancárias. Consulte a política de privacidade do Mercado Pago para mais detalhes.

8. Direitos do usuário

O usuário tem direito a:

  • Acessar os dados armazenados pela extensão (via DevTools do Chrome)
  • Excluir todos os dados desinstalando a extensão
  • Solicitar esclarecimentos sobre o uso dos dados pelo email contato@loggzap.com.br

9. Alterações nesta política

Esta política pode ser atualizada periodicamente. A data de última atualização será sempre exibida no topo desta página. O uso continuado da extensão após alterações implica aceitação da política atualizada.
