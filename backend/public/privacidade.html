📋 O que é o LoggZap

O LoggZap é um aplicativo de automação de mensagens WhatsApp integrado à plataforma Nuvemshop. Ele envia notificações automáticas aos clientes das lojas sobre confirmações de pagamento, atualizações de rastreio e recuperação de carrinho abandonado.

O operador dos dados é o lojista que instalou o LoggZap em sua loja. O LoggZap atua como suboperador, processando dados apenas para cumprir as funções descritas nesta política.

📦 Quais dados coletamos

  • Número de telefone dos clientes da loja (para envio de mensagens via WhatsApp)
  • Número do pedido e código de rastreio (para notificações de entrega)
  • ID do checkout abandonado (para recuperação de carrinho)
  • Status de entrega e histórico de notificações enviadas
Não coletamos: nome completo, CPF, endereço, dados bancários, e-mail ou qualquer dado sensível além do necessário para operação.

🎯 Para que usamos os dados

  • Enviar notificações automáticas de pagamento confirmado
  • Informar atualizações de rastreio e status de entrega
  • Recuperar carrinhos abandonados com mensagens sequenciais
  • Alertar o lojista sobre pedidos com prazo de envio em risco
  • Enviar pesquisa de satisfação após entrega

⏱ Por quanto tempo guardamos

  • Registros de mensagens enviadas: 7 dias
  • Histórico de rastreios entregues: 90 dias
  • Registros de carrinho e boleto: 60 dias
  • Token de acesso OAuth da loja: enquanto o app estiver instalado

Após esses prazos, os dados são removidos automaticamente pelo sistema.

🚫 Opt-out — como parar de receber mensagens

Qualquer cliente pode solicitar a exclusão do envio de mensagens automáticas respondendo com uma das palavras:

PARAR · SAIR · STOP · NÃO QUERO · CANCELAR · DESCADASTRAR

O sistema processa o opt-out imediatamente e confirma por mensagem. O número não receberá mais notificações automáticas.

🔗 Compartilhamento de dados

O LoggZap utiliza os seguintes serviços terceiros para operação:

  • Z-API — gateway para envio de mensagens WhatsApp
  • Nuvemshop API — fonte dos dados de pedidos e clientes
  • Railway — infraestrutura de hospedagem (servidores nos EUA)
  • SeuRastreio — consulta de status de rastreio dos Correios

Não vendemos, alugamos ou compartilhamos dados com terceiros para fins publicitários.
